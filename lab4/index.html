How to Build a PC.

  1. Calculate your budget.
  2. Research compatible parts in your price range.
  3. Pick processor, motherboard, graphics card, sound card, storage, case, powersuply.
  4. Choose peripherials; Mouse, Monitor, Keyboard, Webcam, Headphones.
  5. Order parts, and begin assembly.
  6. Build and Test computer.
  7. Install Operating System and software.

How to make a Battleship Game in Atom

  1. Print out a 10x10 grid map
  2. Place the players 5 ships througout the grid, the opponent deploies ships as well outside of each others view.
  3. Players call out coordinates on the grid, guessing their oponent's ships locations.
  4. If the player guesses wrong then the opponent calls out miss, and the game continues.
  5. If a player guesses correct, the opponent calls out hit and marks the ship as hit.
  6. Each ship has a certain aloted hit count.
  7. The game ends when all the ships on a players board have been sunk.
